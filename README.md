# Aroma Identica — Web UI

Edición mejorada: Interfaz Premium oscuro-dorada con micro-interacciones, previsualización en vivo y funciones virales.

Características añadidas:
- Estética premium Dark-Gold con animaciones sutiles.
- Hero cinematográfico con inline SVG de botella (animada).
- Selector de Gama (Estándar / Gama Alta) con tooltip explicativo y animación de precio.
- Previsualización en vivo del nombre del perfume con etiqueta animada y taglines dinámicos.
- Badge sutil "Más vendido" para la versión Estándar.
- Compartir y copiar la descripción del producto (Web Share API / Clipboard fallback).
- Mejores interacciones móviles y micro-animaciones.

Cómo usar:
1. Abrir index.html en un servidor o directamente en el navegador.
2. Escribir el nombre del perfume en el campo y ver la previsualización en tiempo real.
3. Cambiar la Gama para ver el precio actualizarse con animación.
4. Utilizar Compartir/Copiar para generar texto listo para redes.

Notas para desarrolladores:
- Archivos principales: index.html, css/style.css, js/main.js
- .gitignore incluye .DS_Store.
- Se usan APIs modernas (Web Share y Clipboard). Verificar soporte de navegador para funciones avanzadas.
